# LearnLess

An AI-powered learning resource decision tool.

## Problem

Many learners save dozens of articles, tutorials, and courses but struggle to decide where to start.

## Solution

LearnLess compares 3–10 candidate resources against a specific learning goal and recommends:

- Primary recommendation
- Supporting resources
- Not recommended now
- Comparative reasoning
- Confidence level

## Tech Stack

- Next.js
- TypeScript
- Supabase
- DeepSeek API
- Vercel

## Architecture

User Input
→ Resource Parsing
→ AI Recommendation
→ Structured JSON Validation
→ Database Persistence
→ Result Rendering
